# kursach_database

Руководство пользователя
1) Введение
Настоящее руководство пользователя описывает порядок работы с веб-приложением "Фрагмент информационная системы предприятия общественного питания". Приложение предназначено для управления различными аспектами деятельности предприятия общественного питания (кафе, столовая), включая персонал, блюда, продукты, поставки, заявки и отчеты по реализации, с учётом ролевой модели доступа.

2) Страница входа
После открытия адреса сервера приложения в браузере вы попадете на страницу аутентификации. Чтобы войти в систему введите свой логин и пароль и нажмите кнопку "Войти в систему". Если вы забыли свой пароль нажмите кнопку "Забыли пароль? ".

3) Восстановление пароля
При нажатии на кнопку "Забыли пароль?" вас перенаправит на форму ввода почты для восстановления пароля. При корректном вводе своей почты на нее будет отправлен новый пароль для входа.

4) Домашняя страница
При вводе своего логина и пароля на странице аутентификации вас перенаправит на домашнюю страницу. На основной части страницы описаны основные возможности и ролевая модель доступа. В верхней части расположено меню, чтобы перейти в пункт меню кликните по нему.

5) Панель управления
После нажатия на кнопку "Перейти в панель управления" вас перенаправит на страницу "Панель управления". На странице описана информация о пользователе с кнопками "выйти из системы" и "сменить пароль", быстрая статистика с количеством записей для основных таблиц, таблицы данных, которые вам доступны, справочники, а также список последних действий пользователей.

6) Просмотр доступных таблиц
После нажатия кнопки "Перейти" в блоке "Таблицы данных" вас перенаправит на страницу со всеми доступными таблицами. На каждой карточке таблицы описано количество записей в ней, кнопка "Посмотреть таблицу" и "Добавить запись".

7) Просмотр содержания таблицы
После нажатия на кнопку "Просмотреть таблицу" вас перенаправит на страницу просмотра данной таблицы. В нижней части таблицы описаны возможности вашей роли (Добавление Изменение и Удаление), если этих возможностей нет, там будут стоять кресты.
На странице доступен поиск по шаблону, чтобы его использовать введите шаблон и нажмите кнопку "Поиск". Чтобы отменить поиск по шаблону нажмите кнопку "Отчистить".
На странице доступна сортировка по столбцу. Чтобы применить сортировку по убыванию нужно нажать на заголовок нужного столбца. Чтобы применить сортировку по возрастанию нужно нажать на заголовок столбца еще раз. 
На странице доступно ограничение по количеству записей на одной странице (по умолчанию 10 записей), чтобы выбрать другое количество нужно кликнуть по плашке  и выбрать другое количество записей из предложенных. Чтобы перейти к другим записям снизу есть пагинатор с выбором страницы.
Для каждой записи есть возможность редактирования и удаления (самый правый столбец), при условии, что у вашей роли есть доступ к этим действиям.
На странице доступна кнопка "Добавить запись", при условии, что у вашей роли есть доступ к этому действию.

8) Добавление новой записи
При нажатии на кнопку "Добавить запись" вас перенаправит на форму заполнения данных для новой записи. После корректного заполнения всех полей нажмите на кнопку "Сохранить" чтобы создать новую запись и перейти обратно к таблице, либо нажмите кнопку "Сохранить и добавить еще" чтобы создать новую запись и перейти к чистой форме заполнения.

9) Редактирование записи
При нажатии на кнопку "Редактировать запись" вас перенаправит на заполненную форму редактирования. После корректного заполнения всех полей нажмите на кнопку "Сохранить" чтобы сохранить изменения в записи, либо нажмите кнопку "Удалить", чтобы удалить данную запись.

10) Удаление записи
При нажатии на кнопку "Удалить" вас перенаправит страницу подтверждения удаления записи. Нажмите на кнопку "Удалить" чтобы подтвердить удаление записи, либо нажмите кнопку "Отмена", чтобы отменить удаление и перейти обратно к таблице.

11) Страница документы
При нажатии на кнопку "Документы" в меню вас перенаправит на страницу для ввода SQL-запросов. На странице доступно поле для ввода SELECT запроса, описаны доступные таблицы для данного пользователя, а также заготовлены 3 шаблона запросов, в зависимости от вашей роли.
После ввода SQL запроса есть возможность экспортировать таблицу в xlsx-файл.
