# kursach_database

Руководство пользователя
1) Введение
Настоящее руководство пользователя описывает порядок работы с веб-приложением "Фрагмент информационная системы предприятия общественного питания". Приложение предназначено для управления различными аспектами деятельности предприятия общественного питания (кафе, столовая), включая персонал, блюда, продукты, поставки, заявки и отчеты по реализации, с учётом ролевой модели доступа.

2) Страница входа
После открытия адреса сервера приложения в браузере вы попадете на страницу аутентификации. Чтобы войти в систему введите свой логин и пароль и нажмите кнопку "Войти в систему". Если вы забыли свой пароль нажмите кнопку "Забыли пароль? ".
