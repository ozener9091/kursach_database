Общая информация

Система управления предприятием общественного питания предоставляет комплексные возможности для ведения учета, управления и анализа бизнес-процессов.

Основные разделы системы

Управление блюдами

Создание, редактирование и удаление блюд, управление ингредиентами и ценами.

Управление продуктами

Учет остатков, поставки, заявки на продукты и контроль запасов.

Управление персоналом

Учет сотрудников, должностей, трудовых книжек и кадровых данных.

Отчеты

Формирование отчетов по реализации, поставкам и другим показателям.

Поиск и фильтрация

Все таблицы поддерживают поиск по текстовым и числовым полям. Для поиска введите значение в поле поиска и нажмите кнопку "Поиск".

Работа с данными

  • Просмотр данных: Навигация по страницам, сортировка, фильтрация
  • Добавление записей: Используйте кнопку "Добавить запись" в соответствующей таблице
  • Редактирование: Кнопка "Изменить" в колонке "Действия"
  • Удаление: Кнопка "Удалить" в колонке "Действия"

Права доступа

Система использует ролевую модель доступа:

  • Директор: Полный доступ ко всем данным
  • Менеджер: Доступ к блюдам, продуктам, поставкам, заявкам
  • Шеф-повар: Доступ к блюдам, ингредиентам, продуктам
  • Менеджер по кадрам: Доступ к персоналу и трудовым книжкам

Документы

Доступна страница для выполнения SQL SELECT запросов с ограничениями по доступным таблицам и возможностью экспорта в Excel.
